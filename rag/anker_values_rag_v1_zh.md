# Anker Values RAG v1 (ZH)

- Retrieval unit: section-level chunk.
- Required fields per chunk:
  - `chunk_id`
  - `结论`
  - `证据`
  - `适用场景`
  - `citations`
  - `related_rules`
- Answering should hit both:
  - knowledge chunk(s) in this file
  - rule chunk(s) from `rules_contract_v1_zh.md`
