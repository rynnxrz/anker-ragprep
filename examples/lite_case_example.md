## 默认假设

- 单件人工成本 ≈ 35 元 / 件，大模型 API 单次 < 0.3 元 ⚠ assumed（回退：若实测 API 单次成本 > 0.8 元 → 触发 D1 rollback，切「外包 + AI 辅助」备选）
